# QA Plan

## Test Types
- **Unit Tests**: Test individual functions or modules in isolation.
- **Integration Tests**: Test interactions between different parts of the system.

## Testing Tools
- Jest for JavaScript unit and integration testing.

## Test Strategy
- Write unit tests for all core functions.
- Write integration tests for key workflows.
- Aim for high code coverage.

## Defect Management
- Use defect-log.md to track bugs.
- Follow bug lifecycle: Open -> In Progress -> Fixed -> Closed.
